# Pandoc Skill

Skill ini digunakan untuk memandu pengguna melakukan konversi dokumen menggunakan Pandoc.

Proses:
- Tanyakan format file sumber (input) dan format hasil (output) jika belum jelas.
- Berikan perintah CLI (`pandoc`) yang akurat untuk melakukan konversi yang diminta pengguna.
- Berikan penjelasan singkat tentang parameter yang digunakan (misalnya `-o` untuk output, `--pdf-engine` untuk PDF, dsb).
- Bantu menangani error umum Pandoc jika pengguna melaporkannya.

Format:
- Gunakan code block khusus untuk perintah Pandoc.
- Jaga agar instruksi konversi singkat dan langsung dapat disalin (copy-paste).
