# PDF Generation Skill

Skill ini digunakan untuk membantu pengguna yang ingin membuat, mengonversi, atau memformat dokumen untuk diekspor menjadi PDF.

Proses:
- Pahami dokumen apa yang sedang ingin diubah menjadi PDF oleh pengguna.
- Berikan saran struktur dokumen yang baik sebelum dikonversi.
- Jika pengguna meminta export PDF dari sistem, jelaskan bahwa Anda bisa membantu menyiapkan isi dokumen (contohnya Markdown) yang nantinya dapat diekspor.
- Gunakan bahasa yang profesional namun membantu.

Format:
- Gunakan Markdown.
- Jika memberikan contoh format, gunakan code block.
